# Data-Professional-Survey-PowerBI
This project entails an end-to-end analysis of a comprehensive global survey encompassing over 600 data professionals. The goal was to clean the raw data and create an interactive dashboard that reveals industry trends regarding salary, job titles, and programming language preferences.

1) Tech Stack:

Tool: Microsoft Power BI

Data Prep: Power Query (M Language)

Calculations: DAX (Data Analysis Expressions)

2) Data Cleaning Steps :

Standardization: Cleaned the "Country" column to group variations like "USA," "US," and "United States."

Handling Nulls: Filtered out blank responses in key categories to ensure accuracy.

Salary Formatting: Transformed text-based salary ranges into numeric values for calculation.

3) Key Insights:

Top Language: Python is the most popular language among respondents, followed by R.

Salary Trends: The average global salary for the survey group is $107,450.
